Frequently Asked Questions

Everything you need to know about Apex Client

Is Apex Client free?

Yes! Apex Client is completely free to download and use. You just need a valid Minecraft account.

Is it safe to use?

Absolutely. Apex Client is built on Prism Launcher, an open-source project. We don't collect any personal data beyond what's needed for the user detection feature.

Can I use it on servers?

Yes! Apex Client works on any Minecraft server. The user detection feature lets you see other Apex Client users on the same server.

Will I get banned?

Apex Client is a legitimate launcher that doesn't modify gameplay or give unfair advantages. It's safe to use on most servers.

How do I install additional mods?

Right-click your instance and select "Edit". You can add mods from Modrinth or CurseForge directly from the launcher.

What Minecraft versions are supported?

Currently, the Apex Client mod is designed for Minecraft 1.21.1 with Fabric. You can still play other versions without the mod features.
